## Claude Code

Blueprint is distributed as a Claude Code plugin. The active workflows live in `skills/<name>/SKILL.md`.

- Invoke plugin skills as `/blueprint:spec`, `/blueprint:plan`, `/blueprint:implement`, `/blueprint:tdd`, `/blueprint:review`, `/blueprint:browser-verify`, `/blueprint:explain-visually`, `/blueprint:excalidraw-diagram`, `/blueprint:compress`, `/blueprint:branch`, or `/blueprint:commit`.
- `browser-verify` requires Chrome DevTools MCP.
- Keep shared repo guidance in `AGENTS.md`; keep Claude-specific adapter notes here.
