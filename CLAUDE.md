## Claude Code

Blueprint is distributed as a Claude Code plugin. The active workflows live in `skills/<name>/SKILL.md`.

- Invoke plugin skills as `/blueprint:spec`, `/blueprint:plan`, `/blueprint:implement`, `/blueprint:tdd`, `/blueprint:review`, `/blueprint:compress`, `/blueprint:branch`, or `/blueprint:commit`.
- `/blueprint:build` is a deprecated compatibility alias; prefer `/blueprint:implement`.
- Keep shared repo guidance in `AGENTS.md`; keep Claude-specific adapter notes here.
